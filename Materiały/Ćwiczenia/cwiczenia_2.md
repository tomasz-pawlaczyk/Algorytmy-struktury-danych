# Ćwiczenia nr 2

## Kolejka priorytetowa

Kolejka priorytetowa to struktura danych, która zawsze zwraca element o najwyższym (lub najniższym) priorytecie. Najczęściej implementowana na kopcu binarnym, dzięki czemu operacje `insert` i `delete_max` działają w czasie O(log n), a dostęp do elementu o najwyższym priorytecie jest O(1).

|                        | Insert | Delete_max |
| ---------------------- | ------ | ---------- |
| lista nieuporządkowana | O(1)   | O(n)       |
| lista uporządkowana    | O(n)   | O(1)       |
